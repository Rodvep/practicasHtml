GLOSARIO DE TERMINOS

IDE:
Es una aplicación de software que proporciona herramientas completas para los desarrolladores de software para escribir, depurar y probar su código. Incluye un editor de código fuente, herramientas de compilación y depuración, y a menudo un entorno gráfico para facilitar el desarrollo.
PYTHON:
Es un lenguaje de programación de alto nivel, interpretado y de propósito general. Es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes y utilizado ampliamente en desarrollo web, ciencia de datos, inteligencia artificial, y más.
SOFTWARE:
Se refiere a los programas y otros sistemas operativos que se utilizan en una computadora. Incluye aplicaciones, sistemas operativos, utilidades y cualquier otro tipo de código que hace que el hardware de una computadora realice tareas específicas.
INTERNET:
Es una red mundial de computadoras interconectadas que utilizan el protocolo TCP/IP para comunicarse. Facilita la comunicación y el intercambio de información a través de sitios web, correos electrónicos, redes sociales, y otros servicios en línea.
MODELO OSI:
Es un modelo conceptual que estandariza las funciones de una red de telecomunicaciones o un sistema informático en siete capas: Física, Enlace de Datos, Red, Transporte, Sesión, Presentación y Aplicación. Cada capa tiene funciones específicas y se comunica con las capas adyacentes para permitir el flujo de datos de extremo a extremo en una red.
